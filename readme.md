# Start Template
**Шаблон проекта для быстрого старта**


## Старт проекта

### Склонируй репозиторий и перейди в папку проекта
`git clone git@github.com:asevirinov/start-template.git new-project && cd new-project`

### Установи модули
`npm install`

### Запуск с отслеживанием изменений
`npm run start`

### Сборка в папку `dist`
`npm run build`
